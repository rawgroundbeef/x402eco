Modify EcosystemSection.tsx to render logos in ecosystem entry cards:

2. In the entry card (the `<a>` element), add logo rendering before the text content:

   Current structure:
   ```tsx
   <div className="flex items-start justify-between">
     <div>
       <h4>...</h4>
       <p>...</p>
     </div>
     <svg>...</svg>

   ```

   New structure with logo:
   ```tsx
   <div className="flex items-start gap-3">
     {/* Logo or fallback */}
     <div className="flex-shrink-0 w-10 h-10 rounded-lg bg-surface flex items-center justify-center overflow-hidden">
       {entry.logo ? (
         <Image
           src={entry.logo}
           alt={`${entry.name} logo`}
           width={40}
           height={40}
           className="object-contain"
         />
       ) : (
         <span className="text-lg font-semibold text-text-muted">
           {entry.name.charAt(0).toUpperCase()}
         </span>
       )}
     </div>
     <div className="flex-1 min-w-0">
       <div className="flex items-start justify-between">
         <h4 className="font-medium text-text group-hover:text-accent transition-colors">
           {entry.name}
         </h4>
         <svg ... />
       </div>
       <p className="text-sm text-text-muted mt-1">
         {entry.description}
       </p>
     </div>
   </div>
   ```

The fallback shows the first letter of the entry name in a styled box, matching the visual design language (rounded corners, muted text).

Use `bg-surface` for the logo container background (from existing Tailwind theme). If that class doesn't exist, use `bg-card/50` or similar.
